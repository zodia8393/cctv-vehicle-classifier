### 실험 5. Adaptive Conf + ROI 2x Upscale (채택)

| 설정 | 총 검출 | 40px 미만 | 비고 |
|------|-------:|---------:|------|
| conf=0.15 (기존) | 74 | 7 | — |
| conf=0.05 | 280 | 57 | 오탐 과다 |
| **Adaptive Filter** | **91** | **24** | **3.4× (오탐 0)** |

```python
if size >= 40 and conf >= 0.15: keep   # 중대형 유지
if size >= 20 and conf >= 0.08: keep   # 소형 우대
if size >= 10 and conf >= 0.05: keep   # 극소형 최대
```

**ROI**: 상단 37% 영역만 2x upscale → 20px 차량이 40px로 확대 (+30~40% 비용).

**핵심 발견**: "모델을 바꾸지 말고 후처리를 바꿔라" — YOLOv8n은 이미 conf=0.05에서 소형 차량 검출 중. 기존 conf=0.15 고정값이 소형 객체를 차단.
