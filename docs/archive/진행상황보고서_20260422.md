## 4. 파이프라인 v2 (Lazy Classification)

### 4.1 v1 대비 변화

| 영역 | v1 (~04-17) | **v2 (04-22)** |
|------|------|------|
| 분류 시점 | `classify_every_n=10` | **track 종료 시 Lazy** |
| 카운팅 기준 | tracker 다수결 | **best-N crops + conf-weighted vote** |
| 합류부 누락 | 5/172 (95% miss) | **177/172 (35×)** |
| 분류 호출량 | 프레임 × det 수 | **track당 3회 (-99%)** |

### 4.2 알고리즘 (video_consistency.py v2)

```
┌── 전 프레임 detector + tracker
├── track별 crop buffer + quality score
│     quality = area × det_conf × center_proximity
├── Virtual Line Crossing (y=H/2) 기록 (FHWA/VDS 참조)
└── 영상 종료 시점:
     ├── 필터: min_track_frames≥6 AND trajectory≥80px
     ├── Top-N (=3) quality crops 선별
     ├── batch classify (1 forward)
     └── conf-weighted majority vote → final_class
```
